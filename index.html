Bennett Machine Learning Group

Bennett ML Group is a community of undergraduates aimed at promoting and fostering the growing interest around machine intelligence on campus.

Join us

Bennett ML Group holds (nearly) weekly reading groups on topics in machine learning, from theory to computer vision to systems applications

Prerequisites

You should have a basic ML background, or a basic software engineering background.
